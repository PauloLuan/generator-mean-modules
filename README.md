## Getting Started

![Yo dawg, I heard you like generators?](http://i.imgur.com/2gqiift.jpg)

```bash
npm install -g yo
```

### Yeoman Generators

To install generator-mean-modules from npm, run:

```bash
npm install -g generator-mean-modules
```

Finally, initiate the generator:

```bash
yo mean modules
```
